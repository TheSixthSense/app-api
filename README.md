# app-api
Veganner Backend Repository 
<br />

## 개요
- 개발기간: 2022.06.27 ~
- 개발인원 : 2명

<br />

## 개발환경
- JAVA11
- Spring Boot
- Spring Security
- Spring Data Jpa
- Querydsl
- MariaDB
- Swagger
<br />

## 주요기능
- **회원**
  - 회원가입/회원탈퇴
  - 로그인/로그아웃
  - 회원정보 확인/수정
- **챌린지**
  - 챌린지 참여
  - 챌린지 인증(이미지/글)
  - 일별 챌린지 조회
  - 월별 챌린지 조회
  - 챌린지 가이드 조회
